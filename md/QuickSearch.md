# 使用方法

1. 先进入设置更改Safari默认搜索为DuckDuckGo；
2. Safari内输入`快捷键`+`空格`+`搜索关键词`快速指定搜索引擎搜索，如指定使用维基百科搜索`TUN`，则直接在Safari地址栏内输入`wk TUN`即可直达维基百科的搜索结果页面；
3. 目前仅内置了一些常用站点的支持，他们的`快捷键`如下：
